# Design System Notes

## Visual hierarchy
- Home: one primary CTA
- Session: one action at a time
- Rest screen: timer is the visual priority

## Rest screen hierarchy
1. Rest timer
2. +/- adjustment buttons
3. Next exercise hint
4. Skip button

## Suggested behavior
- Circular progress around the timer is optional
- Smart color change near the end of rest is allowed
- Rest timer should be centered and visually dominant

## Spacing
- Keep the layout clean with generous whitespace
- Avoid crowding the rest screen
- Maintain a clear touch area for buttons

## Typography
- Large timer font
- Small supporting labels
- Minimal text on active training screens

## Product tone
- Clear
- Calm
- Encouraging
- No noisy gamification
