# Onboarding Screen

## Purpose
Collect only the minimum information needed once per user.

## Questions
- Push-up level
- Pull-up level
- Squat level
- Mobility level

## Design rules
- One question per screen
- Multiple choice only
- No typing
- No long explanations

## Flow
1. Push-up question
2. Pull-up question
3. Squat question
4. Mobility question
5. Save answers
6. Calculate level
7. Navigate to Home

## Output
- User level
- Mobility level
- onboardingCompleted = true
