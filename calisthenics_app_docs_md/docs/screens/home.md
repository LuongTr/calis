# Home Screen

## Purpose
Get the user to start training fast.

## Main goal
One primary CTA should dominate the screen.

## Main sections
### Header
- Greeting
- Optional short status text

### Primary CTA
- Start Workout

### Recommended section
- 2–3 workout cards only
- Main recommendation first
- Alternative second
- Mobility third (optional)

## Layout priority
1. Start Workout button
2. Recommended workout cards
3. Secondary browse options

## Interaction flow
- Tap Start Workout → open the top recommended workout
- Tap a recommended card → open workout detail
- Do not overload with filters on the first screen

## Copy tone
- Ready to train?
- Recommended for you
- Start today’s workout
