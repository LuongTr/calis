# Workout List Screen

## Purpose
Show the workouts available for the current category or recommendation bucket.

## Main elements
- Category title
- Short filter chips if needed
- Workout cards

## Recommended content
- Title
- Duration
- Level
- Type
- Simple label like No equipment or Mobility

## Interaction
- Tap a card to open workout detail
- Keep the list short and curated
- Do not make browsing the main goal of the app
