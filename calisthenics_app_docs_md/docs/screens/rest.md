# Rest Screen

## Purpose
Give the user a focused rest timer between sets and between exercises.

## Visual hierarchy
1. Rest time
2. +/- rest buttons
3. Next exercise hint
4. Skip button

## Main elements
- Large centered rest timer
- Minus button on the left
- Plus button on the right
- Optional next exercise label
- Skip button
- Optional circular progress around the timer
- Optional smart color change near the end of rest

## Behavior
- Rest starts automatically after a set ends
- User can add or remove rest time
- Rest time must have a maximum boundary
- Rest time must not go below zero
- Skip should immediately advance the flow

## Layout rule
The rest timer should visually dominate the screen.
All other elements must stay smaller and secondary.
