# Workout Detail Screen

## Purpose
Let the user inspect a workout before starting it.

## Main elements
- Workout title
- Duration
- Level
- Summary tags
- Exercise list
- Start Workout button

## Exercise list item contents
- Exercise name
- Reps or time
- Sets
- Rest target

## Interaction flow
1. User opens workout detail
2. User reviews the workout
3. User taps Start Workout
4. Session screen starts

## Optional
- Small video preview icon per exercise
- No autoplay here in MVP
