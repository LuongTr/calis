# Auth Screen

## Purpose
Let the user enter the app quickly.

## Main elements
- App logo/title
- Short value statement
- Continue with Google button
- Continue with Email button
- Login link

## Layout
Top:
- Logo

Center:
- App message

Bottom:
- Primary auth buttons

## Workflow
1. User opens app
2. User signs up or logs in
3. First-time user goes to onboarding
4. Returning user goes to Home

## CTA rules
- Keep only 1–2 primary actions
- Avoid clutter
