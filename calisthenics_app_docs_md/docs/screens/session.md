# Session Screen

## Purpose
This is the core training experience.

## Main state
- Active set view

## Main elements
### Top
- Embedded YouTube video or small preview player

### Middle
- Exercise name
- Reps or time
- Set counter

### Bottom
- Done button

## Workflow
1. Show current exercise and set
2. User completes the set
3. User taps Done
4. App transitions to Rest screen
5. After rest, app continues to next set or next exercise

## Rules
- Video is support only
- Keep the screen focused on the current set
- Do not show too much text
- Do not require the user to log reps manually in MVP
