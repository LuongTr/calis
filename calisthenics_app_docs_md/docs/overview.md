# Product Overview

## Product idea
A guided at-home workout app focused on calisthenics as the main category and mobility/stretching as the support category.

## Core promise
The app helps a user open the app, get a recommended workout, and start training immediately with minimal thinking.

## Main principles
- Guided, not tracker-first
- Minimal user decisions
- One primary CTA on Home
- Session flow is the core experience
- YouTube video is support, not the main content

## MVP scope
### Included
- Sign in / sign up
- One-time onboarding
- Level estimation
- Workout recommendations
- Preset workouts
- Session flow with rest countdown
- Rest screen controls
- Complete screen

### Excluded
- Calorie tracking
- Workout history tracking in detail
- Custom workout builder
- PPL split in phase 1
- AI-driven recommendations
- Social features

## User types
- Beginner
- Intermediate

## Training style in phase 1
- Beginner: Full Body
- Intermediate: Upper / Lower

## Future phase
- Push / Pull / Legs
- Custom split selection
- Custom workouts
- Smarter adaptation logic
