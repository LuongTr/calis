# Workout Templates

## Template goals
- Simple to follow
- Adapted to level
- Compatible with set → rest → next flow
- Preset only for MVP

## Level rules
### Beginner
- Default to Full Body
- Shorter sessions are allowed, but keep them purposeful

### Intermediate
- Default to Upper / Lower
- Main sessions should be 25 minutes or more
- Short sessions can still exist as light options, but they should not be the primary recommendation

## Template structure
A workout template should contain:
- title
- level
- type
- duration
- warm-up
- main section
- optional cooldown
- list of exercise blocks

## Recommended MVP templates
### Beginner
- Full Body 10 min
- Full Body 15 min
- Mobility 10 min

### Intermediate
- Upper Body 25 min
- Lower Body 25 min
- Upper Body 30 min
- Lower Body 30 min
- Mobility 10 min

## Template design rules
- Do not use circuit logic in MVP
- Keep workout flow linear
- Avoid too many exercises in one session
- Make sure each template maps cleanly to the session state machine
