# Calisthenics Guided Training App Docs

This package contains the product/spec documentation for the app.

## How to use
Read in this order:
1. `docs/overview.md`
2. `docs/user-flow.md`
3. `docs/state.md`
4. `docs/session-state-machine.md`
5. `docs/workouts/templates.md`
6. `docs/workouts/exercises.md`
7. `docs/api.md`
8. `docs/screens/*.md`

## Current MVP scope
- Guided home workout app
- Categories: Calisthenics (main) + Mobility/Stretching (support)
- Onboarding only once per user
- Recommendation rules + simple rotation
- Preset workout templates only
- Session flow with set → rest → next
- YouTube embed only in session screen

## Not in MVP
- Calorie tracking
- Custom workout builder
- PPL split
- Social/community
- AI coach
